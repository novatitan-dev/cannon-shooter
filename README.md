## ✨ Features
- **Tactical HUD Online**: Aerospace-inspired UI with real-time score, health, and power tracking.
- **Dynamic Environments**: Switch between **Daylight** and **Night/Dark** views in the settings.
- **Power-up System**:
  - 🛡️ **Shield**: Temporary invincibility and ability to destroy balls on contact.
  - ⚡ **Double Fire**: Fire two bullets at once for maximum destruction.
- **Progressive Difficulty**: Spawn rates and ball health increase as you survive longer.
- **Mobile Optimized**: Smooth swipe-to-move controls and responsive layout.

## 🕹️ How to Play
1. **Move**: Swipe left or right (or use your mouse) to position the cannon.
2. **Shoot**: Double tap and hold (or click and hold) to fire.
3. **Power-ups**: Collect falling Shield (🛡️) and Double Fire (⚡) icons.
4. **Survive**: Don't let the balls hit your cannon!

## 🛠️ Technical Details
- **Engine**: Pure Vanilla JavaScript (No heavy frameworks).
- **Rendering**: HTML5 Canvas API.
- **Styling**: Tailwind CSS & Modern CSS Gradients.
- **Audio**: Custom Web Audio API implementation for low-latency sound effects.
