# 🚀 Cannon Shooter - Tactical Precision

A high-performance, aerospace-inspired 2D arcade shooter built with vanilla JavaScript and HTML5 Canvas.

## 🎮 Play the Game
**[Live Demo on GitHub Pages](https://novatitan-dev.github.io/cannon-shooter/)**

---

## ✨ Features
- **Tactical HUD Online**: Aerospace-inspired UI with real-time score, health, and power tracking.
- **Dynamic Environments**: Switch between **Daylight** and **Night/Dark** views in the settings.
- **Personalized Profile Dialog**: Editable player names saved locally, keeping user data private.
- **Interactive Popup Panels**: Premium glassmorphic popups for **How to Play** instructions and profile dashboard.
- **First-Time User Consent**: Clean, secure Privacy & Terms overlay to comply with store policies.
- **Power-up System**:
  - 🛡️ **Shield**: Temporary invincibility and ability to destroy balls on contact.
  - ⚡ **Double Fire**: Fire two bullets at once for maximum destruction.
- **Progressive Difficulty**: Spawn rates and ball health increase as you survive longer.
- **Mobile Optimized**: Smooth swipe-to-move controls and responsive layout.

## 🕹️ How to Play
### Mobile Controls
1. **Move**: Swipe left or right to position the cannon.
2. **Shoot**: Double tap and hold on the screen to fire.
3. **Power-ups**: Collect falling Shield (🛡️) and Double Fire (⚡) icons.

### Desktop Controls
1. **Move**: Use **A/D** keys or **Left/Right Arrow** keys.
2. **Shoot**: Press **F** to fire.
3. **Survive**: Don't let the balls hit your cannon!

## 🛠️ Technical Details
- **Engine**: Pure Vanilla JavaScript (No heavy frameworks).
- **Rendering**: HTML5 Canvas API.
- **Styling**: Tailwind CSS & Modern CSS Gradients.
- **Audio**: Custom Web Audio API implementation for low-latency sound effects.
