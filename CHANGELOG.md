# Changelog - Cannon Shooter

## [3.0.0] - 2026-07-09
### Added
- **Challenge Mode (Option B)**: A premium grid selection containing four unique challenge operations.
  - **Operation Time Attack**: Destroy 30 balls as fast as possible.
  - **Operation Iron Dome**: Survive 60 seconds against rapid ball spawns with no revives.
  - **Operation Pure Skill**: Survive 45 seconds with all power-ups disabled.
  - **Operation Titan Brawl**: A single massive ball (Titan) spawns. It splits multiple times into smaller, fast balls when damaged, requiring high precision to fully clear.
- **HUD Mode Integration**: Dynamic challenge progress indicators showing remaining time or balls destroyed.
- **Challenge Results Screen**: Clean victory and failure states with performance stats saved in localStorage.

## [2.0.0] - 2026-05-30
### Added
- **Double Fire Mechanic**: Power-up drop (⚡) that allows the cannon to shoot two bullets simultaneously.
- **Dark Mode View**: Added a dark backdrop setting togglable via the settings menu.
- **HUD Timers**: Added badges displaying remaining seconds for active buffs.
- **Player Profile**: Randomized name generator and profile naming dialog.
- **How to Play Overlay**: Responsive modal explaining swipe/keyboard controls.
- **User Consent**: First-time user privacy consensus popup.

## [1.0.0] - 2026-05-24
### Added
- **Classic Mode**: Standard infinite survival gameplay.
- **Audio System**: Web Audio API integration for machine-gun firing and metallic ball impact sounds.
- **Controls**: Swipe controls on mobile, A/D & arrows keys + F on desktop.
